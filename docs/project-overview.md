# ManaTuner - Project Overview

## Executive Summary

**ManaTuner** is an advanced Magic: The Gathering manabase analyzer that uses Frank Karsten's hypergeometric probability mathematics to help players optimize their deck's mana sources. The application provides sophisticated analysis of casting probabilities, mulligan decisions, and land composition.

## Project Purpose

ManaTuner solves a critical problem for MTG players: determining the optimal number and composition of lands needed to consistently cast spells on curve. The tool provides:

1. **Castability Analysis** - Probability of casting each spell on its optimal turn
2. **Mulligan Simulation** - Monte Carlo simulation for mulligan decisions
3. **Mana Source Breakdown** - Detailed analysis of color sources
4. **Land Recommendations** - Data-driven suggestions for manabase optimization
5. **Export Capabilities** - PDF, PNG, and JSON export for sharing analyses

## Key Features

### Core Analysis

- Hypergeometric probability calculations
- Monte Carlo mulligan simulation (10,000+ iterations)
- Multi-color pip requirement analysis
- Turn-by-turn probability curves

### User Experience

- 100% client-side processing (privacy-first)
- PWA support for offline use
- Dark/Light theme
- Mobile responsive design
- Keyboard shortcuts

### Data Integration

- Scryfall API for card data
- Local storage persistence (AES-256 encrypted)
- Multiple export formats (MTGA, Moxfield, etc.)

## Architecture Pattern

The application follows a **Component-Based Architecture** with:

- Custom hooks for business logic abstraction
- Service layer for calculations and API calls
